# Kayla Robertson — Capstone Project

**Howard University | MS Applied Data Science and Analytics**

This repository contains the statistical analysis and data preparation notebook for my graduate capstone project: the Health Impact Score, a composite nutrition scoring system that ranks 7,500+ foods from the USDA FoodData Central database on a 0 to 1 scale.

## Files

- `Health_Impact_Score_Statistical_Analysis.ipynb` — Full EDA, hypothesis testing, VIF multicollinearity analysis, and pre-ML data preparation
